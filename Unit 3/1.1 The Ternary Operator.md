### Ternary Operator

The **ternary operator** is a shorthand way of writing if-else statements. Here is the synyax:

```java
condition ? expressionIfTrue : expressionIfFalse;
```

It's called the ternary operator because "ternary" means it has three parts.

Here are a few examples comparing if-else statements to their equivalent statements using the ternary opertator.

If-else:

```java
String message;
if (number > 0) {
    String message = "Positive";
} else {
    String message = "Non-Positive";
}
```

Ternary Operator:

```java
String message = number > 0 ? "Positive" : "Non-Positive";
```

If-else:

```java
String result;
if (age >= 18) {
    String result = "Adult";
} else {
    String result = "Minor";
}
```

Ternary Operator:

```java
String result = age >= 18 ? "Adult" : "Minor";
```
